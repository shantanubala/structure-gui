Detailed Description

Utility class to exist with creating OCC files from streamed data. An OCC file is a bespoke Occipital file format for recording streamed data from a sensor, allowing Occipital to playback recorded data streams in house as if the same hardware was used.

Write the provided synchronized frames to the OCC file.
