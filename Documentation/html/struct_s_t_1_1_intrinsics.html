◆ closestRoundQVGALikeIntrinsics()

Intrinsics ST::Intrinsics::closestRoundQVGALikeIntrinsics ( ) const

Return intrinsics for an image size as close as possible to 320x240, but making sure it's a round scale factor. For example, calling this on a 648x484 image will return intrinsics for a 324x242 image. This means one can simply transform from the source image to the new image by using a scale factor on each dimension, no offset handling needed.
